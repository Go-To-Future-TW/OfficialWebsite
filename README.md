# 邁向未來 - 官方網站文檔

## Resources
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **CSS**: Tailwind CSS
- **Deploy**: CloudFlare

## 開發環境設定

### Installation
```bash
pnpm install
# or
npm install
# or
yarn
```

### Hot-Reloading

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```
開發伺服器將在 `http://localhost:3000` 啟動。

### Build
```bash
pnpm build
# or
npm run build
# or
yarn build
```

### Production

```bash
pnpm start
# or
npm start
# or
yarn start
```
